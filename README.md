# Tic Tac Toe

## How to Compile and Run
```
cc TicTac.c
./a.out
```

## How to Play

- `X` goes first
- Select a number between 1 and 9 then click enter
- `O` goes next
- Select a number between 1 and 9(Other than number with `X`)
- Keep playing until there is a winner or the game is draw
